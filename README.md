# 📊 Sales Performance & Target Analysis Dashboard (Power BI)

## 📌 Overview
This project focuses on analyzing sales performance, profit trends, and target achievement using an interactive Power BI dashboard.

## 🛠 Tools Used
- Power BI
- DAX
- Excel

## 📈 Key Features
- KPI tracking (Sales, Profit, Profit Margin)
- Target vs Achievement analysis
- Monthly trend analysis
- Category & sub-category performance insights

## 🔍 Key Insights
- Identified top-performing and loss-making products
- Analyzed seasonal sales trends
- Improved business decision-making through data visualization
